 Elaobração de um APP que calcula Alfa de Cronbach é uma medida de confiabilidade que avalia a consistência 
 interna de um conjunto de itens, como as perguntas de um questionário. Ele verifica o quanto os itens de
 uma escala estão relacionados entre si e medem o mesmo construto subjacente, como a satisfação do cliente 
 ou a autoestima. Um valor alto indica que os itens são consistentes, enquanto um valor baixo pode indicar 
 que os itens não estão medindo a mesma coisa ou que são inconsistentes.  
 <br> Como funciona Avaliar a consistência: <br>
 interna: <br>
 O alfa de Cronbach mede o grau em que os itens de uma escala concordam entre si. <br> 
 Indica confiabilidade: <br>
 É uma medida da confiabilidade da escala, ou seja, a extensão em que ela é uma 
 medida consistente de um conceito.  <br>
 Correlação entre itens: <br>
 Ele é calculado com base nas correlações entre 
 os itens da escala. <br>
 Quanto mais correlacionados os itens estiverem, maior será o valor do alfa.  <br>
 Escala de 0 a 1: <br>
 Teoricamente, o valor do alfa varia de \(0\) a \(1\). Valores a partir de \(0,70\) 
 são geralmente considerados aceitáveis, com valores mais altos indicando maior confiabilidade.  <br>
 Limitações: <br>
 O alfa de Cronbach pode ser baixo se houver poucos itens na escala ou se os itens forem muito heterogêneos.
